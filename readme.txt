DigiBlocks is a powerful collection of beautifully designed Gutenberg blocks that help you create stunning WordPress pages with ease.

== Description ==

= 🚀 BEAUTIFUL BLOCKS FOR THE WORDPRESS BLOCK EDITOR =

DigiBlocks enhances your WordPress Gutenberg editor with a suite of professional-grade blocks designed for performance, flexibility, and ease-of-use. Create beautiful content layouts without writing a single line of code.

= 💎 Feature-Rich Block Collection =

DigiBlocks includes a comprehensive collection of blocks designed to address all your content creation needs:

* **Accordion/FAQ Block**: Create collapsible content sections with schema markup for SEO
* **Buttons Block**: Design beautiful call-to-action buttons with multiple styling options
* **Call-to-Action Block**: Create high-converting CTA sections with customizable layouts
* **Countdown Block**: Add dynamic countdown timers for events or promotions
* **Counter Block**: Display animated number counters with icons and descriptions
* **Google Map Block**: Embed customizable Google Maps with multiple markers
* **Heading Block**: Create eye-catching headings with separators and highlight options
* **Icon Block**: Add customizable icons from a vast library with styling options
* **Icon Box Block**: Feature services or products with icons, titles and descriptions
* **Icon List Block**: Create beautiful lists with custom icons and formatting
* **Pricing Table Block**: Design responsive pricing comparison tables
* **Separator Block**: Add stylish dividers with various designs and optional text
* **Social Icons Block**: Display social media links with customizable icons
* **Spacer Block**: Control vertical spacing between blocks with precision
* **Table Block**: Create responsive comparison and data tables with ease
* **Team Block**: Showcase team members with images, bios, and social links
* **Testimonials Block**: Display customer testimonials in various layouts

= 🎯 Theme Builder Elements =

* **Logo Block**: Add a customizable logo with multiple styling options
* **Navigation Block**: Create responsive menus from WordPress or custom menus

== Installation ==

= Quick Start Guide =

1. **Upload & Activate**
   * Visit Plugins > Add New
   * Search for "DigiBlocks"
   * Click "Install Now" and then "Activate"
   * Or upload the plugin files manually to `/wp-content/plugins/digiblocks`

2. **Start Using the Blocks**
   * Edit any page or post with the Block Editor
   * Look for the "DigiBlocks" category in the block inserter
   * Click on any DigiBlocks block to add it to your page
   * Customize using the block controls in the sidebar

= System Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* Modern browser (Chrome, Firefox, Safari, Edge)

== Frequently Asked Questions ==

= Will DigiBlocks work with my theme? =

Yes! DigiBlocks is designed to work with any WordPress theme that supports the block editor. The blocks adapt to your theme's styling while maintaining their functionality.

= Does DigiBlocks slow down my website? =

No. DigiBlocks is built with performance in mind. CSS and JavaScript files are only loaded when blocks are actually used on a page, ensuring minimal impact on page load times.

= Can I use DigiBlocks on multiple websites? =

Yes, you can use DigiBlocks on as many websites as you want.

= Are the blocks responsive for mobile devices? =

Absolutely! All DigiBlocks are fully responsive and include controls to customize their appearance on desktop, tablet, and mobile devices.

= Can I disable blocks I don't need? =

Yes, DigiBlocks includes a block manager in the admin dashboard where you can enable or disable individual blocks to keep your editor clean and focused.

= Does DigiBlocks support Google Fonts? =

Yes, DigiBlocks supports Google Fonts, giving you access to hundreds of font families for your blocks.

= Is DigiBlocks compatible with page builders? =

DigiBlocks is specifically designed for the WordPress block editor (Gutenberg). While some blocks may work with page builders that support Gutenberg blocks, we recommend using DigiBlocks with the native WordPress editor for the best experience.

= Do I need coding knowledge to use DigiBlocks? =

No coding knowledge is required! DigiBlocks is designed to be user-friendly with intuitive controls.
